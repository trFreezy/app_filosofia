Project Title: Philosophy Analytics Hub: NLP for Educational Content

Overview: > Engineered an interactive data application designed to analyze classical philosophical texts using Natural Language Processing (NLP). This tool empowers educators and content creators to evaluate the tonal sentiment (Optimistic, Pessimistic, or Objective) of historical quotes and track student engagement metrics to optimize lesson plans and study materials.

Technologies Used: > * Python (Core logic and algorithm design)

Pandas (Data structure manipulation)

TextBlob / NLP (Semantic analysis and tone classification)

Streamlit (Front-end UI and interactive data visualization)

Key Achievements:

Architected an interactive Streamlit dashboard that filters and visualizes academic data, providing a seamless user experience for educators.

Implemented a tailored NLP algorithm to classify complex philosophical texts into tonal categories, showcasing the ability to adapt AI tools to niche academic contexts.

Integrated visual analytics (bar charts and KPI metrics) to track "Student Bookmarks," simulating real-world engagement tracking to improve curriculum resources and foster critical thinking skills.
